# AuraBank ✦

**The Financial Vibe-Check**

AuraBank is a Gen-Z focused financial wellness prototype built with a stunning **Neo-Brutalism × Cyberpunk** aesthetic. It moves away from traditional, boring banking apps by introducing concepts like an **Aura Score**, **Fin-Bestie AI**, and "Girl Math/Boy Math" calculators to make tracking spending fun, engaging, and highly visual.

## Features
- ✦ **Aura Engine**: Dynamically calculates your financial vibe (e.g., *Glowing Era 🌿* or *Red Flag Arc 🚩*) based on your simulated spending habits.
- 🧮 **Math Modes**: Toggle *Girl Math* or *Boy Math* to get hilarious, culturally relevant justifications for your spending.
- 🧛 **Subscription Slayer**: Quickly identify and "slay" subscriptions that are bleeding your wallet dry, with real-time UI recalculations.
- 🎯 **Squad Goals**: Track shared savings goals with your friends on a competitive, interactive leaderboard.
- 🎨 **Dark/Light Mode**: Easily switch between a sleek Cyberpunk Dark mode and a high-contrast F5F5F0 Light mode.

## Tech Stack
- **Frontend Framework**: Vite + Vanilla JavaScript
- **Styling**: Pure CSS (Custom variables, flexbox/grid, fluid typography)
- **Fonts**: Inter (Body) & Outfit (Headings)

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the local development server: `npm run dev`
4. Visit `http://localhost:5173` in your browser.
